# India Talent Acquisition Dashboard

Real-time hiring metrics dashboard for India operations.

## Features
- Live data from Google Sheets
- Daily auto-updates
- Department and recruiter performance
- Pipeline analysis
- Aging roles alerts

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

Deployed on Vercel

## Environment Variables

Required in Vercel Settings:
- NEXT_PUBLIC_SHEET_FILE_ID
- NEXT_PUBLIC_DRIVE_API_KEY
